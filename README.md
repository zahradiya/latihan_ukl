Website Presensi Online ini dibuat menggunakan HTML, CSS, dan JavaScript sebagai tampilan depan (frontend) dari sistem absensi.
Tujuan utamanya adalah untuk mempermudah proses presensi pengguna secara digital, mulai dari login, mencatat kehadiran, melihat data pengguna, riwayat presensi, hingga analisis kehadiran dalam bentuk grafik.

Desain web dibuat sederhana, interaktif, dan mudah dipahami, sehingga pengguna dapat dengan cepat memahami alur dan fungsinya.
Walaupun saat ini masih menggunakan data simulasi, website ini sudah siap untuk dihubungkan dengan backend API agar dapat menyimpan dan mengambil data secara otomatis dari server.

Secara keseluruhan, proyek ini menunjukkan penerapan dasar pembuatan aplikasi web dengan tampilan menarik dan fungsi yang relevan untuk kebutuhan presensi online di sekolah atau tempat kerja.
